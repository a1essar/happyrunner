HappyRunner

HappyRunner — simple jQuery plugin to create a sliding runner for hover to menu

Features

  • Use css3 transition for best performance
  • Simple customization using css
  • Responsive
  • ie9+, chrome, firefox, safari, mobile safari, android
  • jQuery animate for old browsers

Examples

  •  Hover with line
  •  Hover with background
  •  Hover with fixed width and height
  •  Vertical menu
  •  Vertical menu with fixed height
  •  Hover with different color
  •  Hover with many runners
  •  Callback function after animation complete

Usage

HappyRunner requires jQuery and Modernizr to run. Download latest version of jQuery and Modernizr and place them to your page.

Add HappyRunner to your page.

<link rel="stylesheet" href="happyrunner/happyrunner.css"/>
<script src="happyrunner/jquery.happyrunner.min.js"></script>

Initialization HappyRunner plugin

<script>
$(function() {
    $(".js__happyrunner").happyrunner({
        fixedWidth: false, /* Use a fixed width for the runner */
        fixedHeight: false, /* Use a fixed height for the runner */
        outerWidth: true, /* Use a outer width element for calculation width the runner */
        outerHeight: true, /* Use a outer height element for calculation height the runner */
        runnerCentering: false, /* Centering runner relative element. Valid values: 'horizontal' and 'vertical', default: false */
        animateDelay: 600, /* jQuery animate delay */
        animateEasing: 'swing', /* jQuery easing  */
        callback: function($el) { /* Callback function after animation complete */
            
        } 
    });
});   
</script>

<div class="js__happyrunner happyrunner-nav">
